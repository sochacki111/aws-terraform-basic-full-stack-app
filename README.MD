# Description
My own Terraform implementation of one of the ACloudGuru hands on lab for creating Full stack application  
# Stack
  - S3 Static website hosting
  - API Gateway
  - Lambda function
  - DynamoDB
  - X-Ray tracing

# Requirements
  - terraform
  - AWS account
# Running
1. ```terraform init```  
2. ```terraform apply```    
3. ```./resources/dynamodb/seed-dynamodb.sh```
